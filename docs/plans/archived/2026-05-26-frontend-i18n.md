# Frontend i18n Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Internationalizzare il frontend Next.js per supportare italiano e inglese usando next-intl v4.8.3, con locale persistito via cookie, nessun prefisso nella URL.

**Architecture:** next-intl middleware composto con NextAuth middleware per locale detection (Accept-Language → cookie → default). Pagine spostate sotto `[locale]/` ma URL pulite grazie a `localePrefix: 'never'`. Tutte le stringhe UI migrate in file JSON con namespace annidati.

**Tech Stack:** next-intl 4.8.3, Next.js 14 App Router, NextAuth v5, Tailwind CSS

### Task 2: Create i18n/request.ts

**Files:**
- Create: `frontend/src/i18n/request.ts`

- [ ] **Step 1: Create request config**

```typescript
import { getRequestConfig } from 'next-intl/server'
import { hasLocale } from 'next-intl'
import { routing } from './routing'

export default getRequestConfig(async ({ requestLocale }) => {
  const requested = await requestLocale
  const locale = hasLocale(routing.locales, requested)
    ? requested
    : routing.defaultLocale

  return {
    locale,
    messages: (await import(`../messages/${locale}.json`)).default,
  }
})
```

  params: { locale },

```typescript
import { auth } from '@/lib/auth'
import createMiddleware from 'next-intl/middleware'
import { routing } from './i18n/routing'
import { NextRequest, NextResponse } from 'next/server'

const handleI18n = createMiddleware(routing)

export default auth(async (req: NextRequest) => {
  const { pathname } = req.nextUrl

  if (pathname.startsWith('/api/')) {
    return NextResponse.next()
  }

  return handleI18n(req)
})

export const config = {
  matcher: '/((?!api|trpc|_next|_vercel|.*\\..*).*)',
}
```

### Task 12: Add LocaleToggle component

- Modify (embed in): `frontend/src/components/Sidebar.tsx`

- [ ] **Step 1: Create server action for locale change**

Create `src/lib/locale.ts`:
```typescript
'use server'

import { cookies } from 'next/headers'

export async function setUserLocale(locale: string) {
  const cookieStore = await cookies()
  cookieStore.set('NEXT_LOCALE', locale, {
    maxAge: 60 * 60 * 24 * 365,
    path: '/',
  })
}
```

- [ ] **Step 2: Create LocaleToggle client component**

Create `src/components/LocaleToggle.tsx`:
```typescript
'use client'

import { useLocale } from 'next-intl'
import { useTransition } from 'react'
import { setUserLocale } from '@/lib/locale'
import { useTranslations } from 'next-intl'

export function LocaleToggle() {
  const locale = useLocale()
  const t = useTranslations('locale')
  const [isPending, startTransition] = useTransition()
  const nextLocale = locale === 'it' ? 'en' : 'it'

  return (
    <button
      onClick={() => startTransition(() => setUserLocale(nextLocale))}
      disabled={isPending}
      className="text-sm hover:underline"
      aria-label={t('switch_to')}
    >
      {nextLocale.toUpperCase()}
    </button>
  )
}
```
